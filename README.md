# ZURA-
Zora: Nausea Relief Device                                                                                                                                               
​This project focuses on providing comfort for users experiencing nausea through controlled vibration feedback. 

​How it works:                                                                                                                                                          
​-The device uses a button to cycle through different vibration intensity levels.                                                                                        
​-It utilizes Pulse Width Modulation (PWM) to control the vibration motor via the ledc function.

​Technical Specs:                                                                                                                                                       
​-Controller: ESP32 (implied by ledc functions).                                                                                   
​-Pin 4: Vibration Motor.                                                                                                                                               
​-Pin 33: Control Button
